# Views

stored sql select


## references
- wikipedia: [sql view](https://en.wikipedia.org/wiki/View_(SQL))
- w3schools: [sql view](https://www.w3schools.com/sql/sql_view.asp)

## create
```sql
CREATE VIEW _view_name_ AS  
SELECT _column1_, _column2_, ...  
FROM _table_name_  
WHERE _condition_;
```
## drop
with metadata 
```sql
IF EXISTS(SELECT * FROM sys.views WHERE name = '_view_name_'>) {
	DROP VIEW _view_name_
}
```
