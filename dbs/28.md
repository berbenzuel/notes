## create
```sql
CREATE VIEW _view_name_ AS  
SELECT _column1_, _column2_, ...  
FROM _table_name_  
WHERE _condition_;
```
## drop
with metadata 
```sql
IF EXISTS(SELECT * FROM sys.views WHERE name = 'name'>) {
	DROP VIEW name
}
```
