# Views

stored sql select


## references
- wikipedia: [sql view](https://en.wikipedia.org/wiki/View_(SQL))
- w3schools: [sql view](https://www.w3schools.com/sql/sql_view.asp)

## create
```sql
CREATE VIEW _view_name_ AS  
SELECT _column1_, _column2_, ...  
FROM _table_name_  
WHERE _condition_;
```
## drop
with metadata 
```sql
IF EXISTS(SELECT * FROM sys.views WHERE name = '_view_name_'>) {
	DROP VIEW _view_name_
}
```

# CTE
> Common Table Expression

temporary result set in SQL that you can reference within a single query.

## references
- geeks for geeks: [cte](https://www.geeksforgeeks.org/sql/cte-in-sql/)

## create
``` sql
WITH _cte_name_ AS (
SELECT * FROM _table_name_
) 
--must be used immediately
SELECT * FROM _cte_name_
```

## usage
-   Breaks complex queries into small, reusable parts.
-   Makes queries easier to read and understand.
-   Helps work with hierarchical data using recursion.
